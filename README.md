# El Grown Mindset
Se da entender como la capacidad que tenemos para aprender a aprender,sobre todo de nuestras experiencias y con lo que nos rodea e interactuamo, la mentalidad positiva, comprendiendo que se puede sacar lo bueno de lo malo y usarlo para aprender, un camino donde el aprendizaje es algo primordial, donde el amor se da por el aprendizaje siendo esto que abarca un _paradigma_ muy nuevo.
